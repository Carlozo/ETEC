    Componentes de formatação de texto

    < b > texto em negrito;
    < i > texto em itálico;
    < u > texto sublinhado;
    < font size="tamanho "> tamanho do texto;
    < font color="cor" > cor do texto;
    < body text="#cor"> todo o texto na tag body será dessa cor;

    A tag < font > é uma tag normal, pois necessita que tenha unicio e fim envolvendo um texto, mas podemos notar que ela recebe atributos, como color e size.


    Atributo font, < font color="#cor" face="nome da fonte" size="tamanho >, onde color é a cor, face a fonte que será utilizada e size o tamanho da mesma

    Imagem

    Nessa tag podemos definir tamanho, alinhamento, borda e etc.

    EXEMPLO

    < img src="foto.jpg" >, onde .jpg é o tipo de imagem
    < img src="c:/foto.jpg" >, onde c é o destino da imagem


    Nesses exemplos a imagem irá abrir em tamanho real. Para corrigir isso, há dois atributos HEIGHT e WIDTH, que significam altura e largura


    EXEMPLO

    < img src="foto.jpg" height="120" width="120" >

    Podemos também, posicionar apenas a imagem.


    Esta foto < img src="foto.jpg" align="right" width="200" width="200" > alinha a figura a direita
